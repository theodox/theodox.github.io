Key Input Example

  1. Create a new C# script in the project window (Assets > Create > C# Script)
  2. Name it KeyColor
  3. Open it in MonoDevelop
  4. Add two fields inside the class (above Start):
    public Color BaseColor;
public ColorKey[] ColorKeys;
  5. ColorKey is a new class - it doesn’t exist in Unity (and won’t show up in your auto-complete). We’ll add it by going below the body of the class (after the last curly) and adding 
    [System.Serializable]
public class ColorKey
{
    public KeyCode Code;
    public Color Color;
}
    This creates a simple class combining a Unity KeyCode and a Unity Color. The bit in square brackets is an attribute, it tells Unity that it’s OK to store this class on disk. If you make custom classes and want to use them in the editor, you’ll have to mark them with System.Serializable (more details here);
